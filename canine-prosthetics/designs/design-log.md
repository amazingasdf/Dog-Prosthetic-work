# Design Log

Record every meaningful design decision here. One entry per decision. "Why" matters more than "what" — this log is what proves the thinking behind the work.

---

## Template (copy for each entry)

### [Date] — [Decision title]

- **Decision:** (e.g., socket wall thickness set to 4mm)
- **Why:** (what constraint, paper, observation, or failed print drove this?)
- **Alternatives considered:** (what else was on the table and why it lost)
- **How I'll know if it was right:** (what test or print result validates it)

---

### 2026-07-XX — Example: Chose transtibial (below-knee) as the starting configuration

- **Decision:** First prototype targets below-knee amputations
- **Why:** Most common canine amputation site with residual limb; richer literature; simpler load path than above-knee
- **Alternatives considered:** Full-limb cart/wheel systems (already well-solved commercially), carpal supports (less structural challenge)
- **How I'll know if it was right:** Shelter staff interviews confirm demand; v1 print achieves stable fit on a test form
